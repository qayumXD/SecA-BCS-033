# Simple API & Database App

A Flutter app demonstrating:
1. API calls (using JSONPlaceholder API)
2. Data display
3. Username input and save
4. SQLite database storage

## Features
- Fetch user data from REST API
- Display fetched data
- Save username with API data to local SQLite database
- View all saved users

## Run the app
```bash
cd simple_api_app
flutter pub get
flutter run
```
